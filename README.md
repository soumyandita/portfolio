# 🚀 Modern Personal Portfolio Website

A sleek, high-performance, and fully responsive personal portfolio website designed to showcase my journey, technical skill set, and featured full-stack projects as an aspiring web developer.

---

## ✨ Core Features

* **🎨 Persistent Dark/Light Mode:** Built using Bootstrap 5's color modes and native JavaScript `localStorage`. The website remembers the user's theme preference across sessions and page refreshes with zero visual flickering.
* **📱 Ultra-Responsive Architecture:** Engineered with a mobile-first approach using Bootstrap's fluid grid system. Handled container padding and fixed-dimension edge cases to completely eliminate horizontal scrolling or text clipping on small displays.
* **🎬 Interactive Scroll Animations:** Integrated **AOS (Animate On Scroll)** with finely-tuned offset and duration parameters for smooth visual transitions when scrolling up or down.
* **⚡ Modern Component UI:** Features dynamic glowing badge frames, floating custom profile layouts, and micro-interactions on call-to-action buttons.
* **🌐 Cross-Platform Connectivities:** Includes dynamic brand-colored social linking nodes (GitHub, WhatsApp, Telegram, etc.) with custom scaling physics on hover.
